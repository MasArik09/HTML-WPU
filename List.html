Bahasa

HTML
HTML atau HyperText Markup Language adalah bahasa yang digunakan untuk membuat struktur dan konten halaman web di internet.
CSS
CSS, singkatan dari Cascading Style Sheets, adalah bahasa pemrograman yang digunakan untuk mengatur tampilan visual sebuah website, seperti tata letak, warna, dan font. CSS memungkinkan pemisahan konten HTML dari presentasinya, sehingga memudahkan perubahan gaya tanpa mengubah struktur halaman.
JS
JavaScript adalah bahasa pemrograman yang digunakan developer untuk membuat halaman web yang interaktif.
